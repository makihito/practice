Git コマンド
  git add ファイル名
      ファイルを選択する。
  git commit -m "メッセージ"
      選択したファイルを記録する。このように記録することをコミットという

  リモート
      リモートにファイルをアップロードしたり、リモートからファイルをダウンロードすることで、開発者同士がファイルを共有することができる
  git remote add リモート名 リモートURL
      プッシュする先のリモートを登録する
  git push origin master
      登録したリモートにaddしたファイルをアップロードすることができる。（pushするという）
  git pull origin master
      リモートからファイルをダウンロードすることができる。   

  git status
			自分が変更したファイルのファイル名を表示することができる
			どのファイルを add していて、どのファイルがまだ add されていないかを確認することができる.
	git diff
			変更内容を把握する.追加されたコードが緑色で表示される。
	git log
			自分や他人のコミットを確認することができるぞ。
			-pオプション：コミットメッセージだけでなく、変更内容は見れる
